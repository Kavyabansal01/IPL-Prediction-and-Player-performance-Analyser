# 🏏 IPL Analytics & Match Prediction Platform

An AI-powered IPL analytics platform that provides player performance insights, comparison tools, and machine learning-based match prediction using Python and Streamlit.

---

## 📌 Overview

This project is an end-to-end data analytics and machine learning platform built on IPL (Indian Premier League) data.

It allows users to:
- Analyze player performance
- Compare players
- Predict match winners using ML

The application is built using Streamlit and offers an interactive dashboard experience.

---

## 🚀 Features

### 📊 Player Analysis
- Total matches played
- Runs scored
- Strike rate
- Last 10 match performance
- Moving average trend

### ⚔️ Player Comparison
- Compare two players side-by-side
- Runs, matches, strike rate comparison

### 🤖 Match Winner Prediction
- ML-based prediction using Random Forest
- Based on teams and toss
- Displays predicted winner with confidence

### 🎨 Interactive UI
- Sidebar navigation
- Dashboard layout
- Clean and responsive design
